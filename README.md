# C++ If-Else Statement Program
## Description
This is a simple C++ program that demonstrates the use of an `if-else` statement. The program accepts a number from the user and checks whether it is positive or zero/negative.
## Features
- Takes integer input from the user.
- Uses an `if-else` statement for decision-making.
- Displays whether the number is positive or zero/negative.
## Sample Output
### Input
```
10
```
### Output
```
The number is positive.
```
### Input
```
-5
```
### Output
```
The number is zero or negative.
```
## Concepts Used
- `#include <iostream>`
- `cin` and `cout`
- Variables
- `if-else` statements
- Comparison operators
